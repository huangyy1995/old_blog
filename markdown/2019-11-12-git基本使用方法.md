# 常见用法

git commit  
git commit –amend 覆盖上次提交  
git checkout  移动HEAD节点  
git branch  添加分支  
git branch -F  移动分支  
git branch -D  删除分支  
git merge  把node合并到  
git rebase  把当前节点修改应用在node上  
git revert  新建节点revert修改  
git reset  回到老节点  
git cherry-pick … 应用指定node的修改  
git rebase -i  把当前节点修改交互式重排，并应用在node  
git tag  加入tag  
git describe  查看当前node离的最近的tag状态  
git fetch 1. 从远程仓库下载本地仓库中缺失的提交记录 2. 更新远程分支指针(如 o/master)  
git pull 1. git fetch 2. git merge   
git pull –rebase git fetch 2. git rebase
